Flujo de publicación

Solicitud enviada a revisión del admin. Si requiere correcciones, se pausa hasta que el anunciante ajuste la información. Si se autoriza, se habilita la transferencia. Cuando el pago sea confirmado y no existan cambios pendientes, el sistema cambia automáticamente la imagen actual por la imagen aprobada y activa la publicidad.

Pendiente de autorización del administrador

La solicitud queda preparada para que el admin revise datos, imágenes, identificación, datos privados y facturación. Si hay correcciones, el anunciante deberá ajustarlas antes de pagar.

Después de autorizar

Cuando el admin autorice la solicitud, se habilita la transferencia. Si el anunciante solo realiza el pago y no modifica nada más, la publicidad se activa automáticamente y la imagen actual se cambia por la imagen aprobada.
