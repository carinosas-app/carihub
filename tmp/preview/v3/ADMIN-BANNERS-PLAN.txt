# Admin — Imágenes de banners (pendiente de implementar)

## Regla principal
- En HOME, RESULTADOS y PERFIL el visitante/anunciante NO mueve ni edita banners.
- Solo el dueño del sitio (admin) gestiona imágenes y encuadre desde el panel admin.
- Reutilizar lógica base en `js/frame-pan.js` (clic → flechas → arrastrar → clic fijar + lápiz cambiar imagen).

## Entrada en admin
- Botón / menú: **«Imágenes de banners»**
- Pantalla con mapa de posiciones por pantalla y vista previa de la imagen actual en cada slot.

---

## HOME (mockup v4 — pantalla actual)

| ID slot           | Posición              | Formato        | Notas actuales mockup                          |
|-------------------|-----------------------|----------------|------------------------------------------------|
| home-hero         | Arriba (derecha)      | Diagonal/recorte | Carrusel por categoría, fotos `assets/user/` |
| home-ad-centro-izq| Centro izquierda      | Cuadrado       | «Más visibilidad»                              |
| home-ad-centro-der| Centro derecha        | Cuadrado       | «+220%» / crecimiento                          |
| home-ad-inferior  | Abajo (ancho completo)| Rectángulo largo | Rotación 3 slides, `assets/user/*.jpg`       |

Acciones admin por slot:
1. Ver imagen actual
2. Clic en imagen → modo encuadre (flechas ↑↓←→ parpadeando, arrastrar, segundo clic fija)
3. Icono lápiz → subir / reemplazar imagen
4. Guardar `panX`, `panY`, `src` en backend (Firestore o config admin)
